# 📥 00-Inbox

**Швидкі нотатки, ідеї, кліпи з веб**

---

## 📋 Призначення

Це "в'їзна брама" для всього. Сюди потрапляють:
- 🧠 Швидкі думки і ідеї
- 🌐 Кліпи з веб (Web Clipper)
- 📱 Нотатки з мобільного (Obsidian Mobile)
- 💬 Розмови, цитати, посилання
- 📊 Чорнові матеріали перед структуруванням

---

## 📁 Структура

```
00-Inbox/
├── 2026-04-19-идея-название.md
├── 2026-04-18-web-clip-article.md
├── 2026-04-17-quick-thought.md
└── ...
```

**Ім'я файлу**: `YYYY-MM-DD-[коротко-опис].md`

---

## 📝 Формат файлу

```markdown
---
title: "Коротка назва ідеї"
date: 2026-04-19
tags: [inbox, idea, work] # або learning, trading, personal
category: [залежить від тегу]
status: draft
pinecone_indexed: false
---

## 💡 Ідея / Вміст
[Текст, цитата, клід, посилання]

## 🔗 Джерело
[Де це з'явилось - посилання, сторінка, розмова]

## 📌 Теги для подальшої обробки
[Додаткові теги для категоризації]
```

---

## 🔄 Потік роботи

1. **Додайте** файл в Inbox
2. **Потім** (коли є час) **перемістіть** у правильну папку:
   - `10-Work/` — якщо про роботу/кодування
   - `20-Trading/` — якщо про трейдинг
   - `30-Learning/` — якщо навчання/стаття/книга
   - `40-Personal/` — якщо особисте

3. **Очистіть** Inbox або **архівуйте** в `99-Archive/`
