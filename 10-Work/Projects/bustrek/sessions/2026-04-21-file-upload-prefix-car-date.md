## Мета сесії
Доповнити логіку кастомних імен файлів: до імені файлу додати номер авто та дату завантаження. Також `original_name` у зв'язку `files` має зберігати нову назву.

## Виконано

| Задача | Результат |
|---|---|
| Додати номер авто та дату до імені файлу | ✅ суфікс `{номер_авто}_{YYYY-MM-DD}` |
| Порядок частин: назва_авто_дата | ✅ `{custom_name}_{номер_авто}_{дата}.{ext}` |
| `original_name` у `files` = нова назва | ✅ замість сирого клієнтського рядка |
| Витягнути спільну `sanitize()` | ✅ без дублювання транслітерації |
| Fallback якщо транспорт не прив'язаний | ✅ `no_car` замість номера |

## Важливі рішення (ADR)

| Рішення | Причина |
|---|---|
| Порядок: назва_авто_дата (не авто_дата_назва) | Рішення користувача — назва файлу на першому місці |
| `buildPrefix` як окремий метод | Чіткий поділ відповідальності, легко змінити формат |
| `sanitize()` — спільна логіка | Раніше транслітерація дублювалась у `buildFileName` |
| `original_name` = повна нова назва | Щоб у БД зберігалась читабельна назва, а не сирий клієнтський рядок |

## Артефакти

- `app/Services/AppApi/Services/FileUploadService.php` — методи `storeFile`, `buildPrefix`, `buildFileName`, `sanitize`
