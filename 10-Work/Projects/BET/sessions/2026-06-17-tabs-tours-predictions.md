# BET — Майбутні/Архів, групування по турах, прогнози на тури 2-3

## Мета сесії
Реструктурувати веб-застосунок (нові таби + групування по турах) і згенерувати
аналітикою прогнози на всі доступні майбутні матчі ЧС-2026 + експреси.

## Виконано
- **Новий таб «Майбутні»** — лише `pending`-ставки, з акцентом «🔥 Сьогодні» (матчі за `new Date()`), решта згрупована по турах.
- **Новий таб «Архів»** — лише розраховані (`win/lose/void`), фільтри статусу+дати, групування по турах. Показує точність туру (Тур 1 · 59% 19/32).
- **Групування по турах** скрізь:
  - Ставки → панелі «🏟️ Тур N · X ставок · Y%».
  - Матчі → розділювачі «Тур N · зіграно k/24», дата+час у клітинці.
  - Тур виводиться як **matchday** з розкладу: у межах групи 2 матчі = 1 тур (по 24 на тур).
- **Експреси розбито по датах** (заголовки `acca-date-head`) + збережено фільтр-дропдаун.
- **Програш експреса** підкреслено банером «✗ Програв — не зайшла N нога (досить однієї)» (логіка accaStatus вже була: одна нога lose → весь lose).
- **Прогнози аналітика** (агент `football-betting-expert` через general-purpose):
  - 64 нові ставки `pred-026…089`: тур 2 глибоко (2 ставки/матч), тур 3 попередньо (1/матч, note=«попередній»).
  - 5 експресів `agent-acca-6…10` на 19-21.06 (2 high бетон, 2 med, 1 low value).
  - Інтегровано в `ledger/picks.csv` (174) та `ledger/accas.csv` (15), регенеровано `app/seed-data.js` (нова version → авто-імпорт у браузері при reload).
- **E2E**: 34/34 (додано тести Майбутні/Архів/тур-групування/дати-експресів), 0 console-помилок.

## Важливі рішення (ADR)
| Рішення | Чому |
|---|---|
| Тур = matchday з розкладу (2 матчі/групу), а не поле `tour` з CSV | CSV-поле `tour` непослідовне (агентські pred позначені «2», хоча це matchday-1 груп I/J); похідна з розкладу — єдине надійне джерело |
| «Архів» = зіграні; «Ставки» = лише активні (матч не зіграно); «Майбутні» = pending з акцентом «Сьогодні» | За уточненням користувача: у Ставках не показувати старі прогнози, де матч уже закінчився |
| Прогнози без веб-ресерчу | ЧС-2026 — майбутній турнір, веб не дає фактів; агент міркує від сили збірних + результатів т.1 + lessons.md |
