## 🎯 Контекст

Pine proxy для TS-3 Inner FVG Sniper: H1 bias + 15m BOS (fresh ≤30min) + 15m FVG detection + limit entry на FVG proximal edge + SL max 20 pips + RR 1:3. Max 3 trades/day, skip Asia 00-07 UTC, EOD 21:00.

FVG detection algorithmic-friendly: 3-bar gap where `low > high[2]` (bullish) / `high < low[2]` (bearish).

## 🎯 WINNING CONFIGURATION (v5)

```pine
Strategy: ТС-3 v5 — SHORTS only, NO H1 bias, +commission
- TF: 15m EURUSD
- Qty: 10% equity
- Commission: 0.03% per order (percent type)
- Slippage: 2 ticks

Inputs:
- piv_len = 3 (pivots for BOS)
- bos_fresh = 2 bars (≤30 min fresh BOS)
- min_fvg_pips = 5.0
- max_sl_pips = 20.0 (skip wider setups)
- sl_buf_pips = 2.0
- min_rr = 3.0 (TP at 1:3 RR)
- limit_window = 10 bars
- Asia skip: 00-07 UTC
- EOD close: 21:00 UTC
- max_per_day = 3
- Side: SHORTS ONLY
- No H1 bias filter
- No partial TP (full size to final TP)
- No time-stop

Logic:
1. 15m bearish BOS: close < last_pivot_low
2. Within 2 bars of BOS, detect bearish FVG: high < low[2]
3. FVG size ≥ 5 pips, else skip
4. Limit entry at FVG bottom (high of detection bar, proximal edge)
5. SL = max(FVG 50% midpoint, BOS swing high) + 2 pip buffer
6. If SL > 20 pips → skip
7. TP = entry - 3×risk (hard 1:3)
```

## 🔍 Ключові findings

### 1. Time-stop 2h = killer
v1 (time-stop) vs v2 (no time-stop): PF 0.53 → 1.086. Winners not reaching TP 1:3 within 2h at 15m EURUSD.

### 2. Longs are broken, shorts have alpha
v3 breakdown: **Long PF 0.18 (disaster), Short PF 0.91**. H1 bias filter did remove bad longs but also removed good shorts. **Disabling bias + shorts-only** captures 2× sample while preserving edge.

**Hypothesis for asymmetry:** EURUSD structurally bearish during this 5.5-month period (buy-hold only +1.9%); FVG continuations in dominant direction work better than counter-trend.

### 3. Partial TP1 не допомагає
v2 partial close hurt gross profit capture. v4/v5 без partial дали кращий result per trade (30 pips win vs 18 pips).

### 4. H1 timeframe FAIL
v6: 13 trades / 2.3 years. Setups занадто рідкісні на H1 EURUSD.

### 5. Commission impact
v2 → v3: commission 0.03% дропнуло PF 1.086 → 0.448. Але shorts-only зберегло edge за рахунок вищого avg win (30p vs 18p).

## ⚠️ Caveats

- **Sample size 10 trades** — **недостатньо** для statistical significance (TS-3 spec вимагає 40+).
- Період 5.5 міс може бути cherry-picked (shorts-alpha period).
- H1 bias filter disabled у v5 — це deviation від spec, треба переконатися що edge не випадковий.
- Real-world slippage на FVG limit entries часто гірший за backtest.
