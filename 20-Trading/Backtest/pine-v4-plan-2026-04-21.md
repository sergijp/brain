Після V3 (Hybrid на D TF, 5 конфігурацій) отримали **однозначно негативний edge на всіх варіантах**:
- EURUSD/GBPUSD/XAUUSD на D TF: від −16% до −33% P&L

**Рішення:** продовжити Pine testing (не pivot до manual), але з концентрованими гіпотезами, не scatter changes.

---

## 🧪 Три гіпотези

### H1: **H1 timeframe — sweet spot**
- 15m: 22 трейди, +0.23% (малий sample, marginal)
- D: 300+ трейдів, −32% (багато даних, edge негативний)
- H1: не тестовано → ~8000 бар історії на TV Free, між spam і rare
- **Очікування:** якщо edge існує — найімовірніше на H1

### H2: **ATR-based SL** замість swing-based
- Поточний SL: sweep extreme → 50-150 pips wide
- 2R target = 100-300 pips move після entry → рідко
- ATR SL (1.5× ATR14): consistent RR, typical 20-40 pips на H1
- **Trade-off:** втрачаємо SMC stop-hunt protection, але отримуємо консистентну RR

### H3: **Sweep quality filter**
- Зараз: будь-який wick + close above = sweep
- Слабкі sweeps створюють false BOS
- Фільтр:
  - Pierce ≥ 5 pips нижче pivot
  - Close-back ≥ 50% від pierce distance
  - Опц.: volume на sweep > avg × 1.5

## 📋 Критерії переходу до наступних кроків

### Якщо V4 дає позитивний edge на H1:
1. Test на GBPUSD H1 + XAUUSD H1 (robustness)
2. Add session KZ filter (London + NY)
3. Paper forward test 1 місяць
4. Go-live decision за playbook v2 criteria

### Якщо V4 знову негативний edge:
1. **Прийняти negative evidence** — algorithmic SMC не працює на Pine
2. **Pivot до manual Bar Replay** (Варіант B)
3. Manual replay дасть edge з discretionary judgment де Pine не може

### Якщо V4 marginal (close to 0):
1. Test alternative approaches:
   - ORB (Opening Range Breakout) baseline
   - RSI mean-reversion
   - MA crossover з ATR filter
2. Compare SMC edge vs baselines

## 🔧 Технічні нотатки

**Pine code base:** V3 Hybrid (див. `pine-mvp-2026-04-21.md` V3 section)

**Зміни для V4:**
```pine
// H1 (change chart TF)
// ATR SL
atr_len = input.int(14, "ATR length")
atr_mult = input.float(1.5, "ATR SL multiplier")
atr_val = ta.atr(atr_len)
// sl = bl_sw_low - buf  // OLD
sl = close - atr_val * atr_mult  // NEW for longs

// Sweep quality
min_pierce_pips = input.float(5.0, "Min sweep pierce pips")
min_closeback_pct = input.float(50, "Min close-back %")
pierce_dist = last_pl - low  // for bullish sweep
closeback_dist = close - low
sweep_quality_ok = pierce_dist >= min_pierce_pips * pip_sz and closeback_dist >= pierce_dist * (min_closeback_pct / 100)
```

- Chart: EURUSD H1
- Commission: $5/order (~1 pip)
- Slippage: 2 mintick
- Initial capital: $10,000
