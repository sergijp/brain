## 🎯 Контекст і відмінність від ТС-1/2/3

NS — **власна** SMC стратегія (незалежна від `smc-playbook-v2`). Порівняно з ТС-1 (reversal at POI) та ТС-3 (inner FVG) — NS має **масштабованіший каркас**: top-down bias (D1+H4+H1), AMD-session-driven entries, opens як якорі, 5-крокова воронка + 6-point scoring-driven position sizing.

| Елемент | ТС-1 Pine | NS Pine |
|---------|-----------|---------|
| HTF bias | H1 EMA (fast/slow) alignment | **D1 EMA50 slope + H4 EMA50/200 + H1 pivots** ("2 of 3" agreement) |
| POI | algorithmic (last pivot + sweep) | algorithmic + **H1 FVG zone check** |
| Sweep | pierce ≥ 5 pips + closeback ≥ 50% | sweep same + **bonus якщо зачепив open (NYM/WO/MO)** |
| BOS | pivot high/low break | pivot + **session_id tracking для "sweep+BOS same session" scoring** |
| Session | London+NY UTC windows | **Kyiv TZ AMD windows:** Asia 02-08, London 09-12, NY 14:30-19 (via `"Europe/Kiev"` tz) |
| Entry | limit retest at BOS level | FVG proximal / OB wick **(3-bar imbalance detection)** |
| Sizing | fixed 1% | **score-driven: 1% full (≥5) / 0.5% half (3-4) / skip (<3)** |
| SMT | — | **GBPUSD correlation + DXY inverse divergence check** |
| Daily profile | — | **trending/consolidation via ATR20D ratio** — consol day = hard skip |
| RR gate | 1:3 hard | 1:3 hard + **+1 score if RR ≥ 1:5** |
| Risk guards | EOD 21 UTC | EOD 21 UTC + **daily stop -3% + 2 consec loss → pause 1h** |

**Мета:** перевірити чи **сумарний** NS engine (всі 5 кроків + scoring + AMD) дає PF > 1.3 на EURUSD 15m — до того як інвестувати 1-2 тижні у Manual Bar Replay (Етап 1).

## 💻 Pine Script — NS v1

```pine
//@version=6
strategy("NS — Top-Down SMC (5-step funnel + 6-pt scoring)",
     overlay=true,
     initial_capital=10000,
     default_qty_type=strategy.percent_of_equity,
     default_qty_value=10,
     pyramiding=1,
     slippage=2,
     commission_type=strategy.commission.percent,
     commission_value=0.03,
     calc_on_every_tick=false,
     process_orders_on_close=true)

// ═══════════════════════════════════════════════════════════════
// ░ BLOCK 1 — CONSTANTS & HELPERS
// ═══════════════════════════════════════════════════════════════
pip  = syminfo.mintick * 10
tz   = "Europe/Kiev"

f_bar_time_minutes_kyiv() =>
    hour(time, tz) * 60 + minute(time, tz)

// ═══════════════════════════════════════════════════════════════
// ░ BLOCK 2 — INPUTS
// ═══════════════════════════════════════════════════════════════
gS = "1. Structure Detection"
piv_len       = input.int(5,    "Pivot length",                 2, 20, group=gS)
sweep_lb      = input.int(30,   "Sweep lookback bars",          5, 100, group=gS)
min_pierce    = input.float(5.0,"Min sweep pierce (pips)",      1, 50, step=0.5, group=gS)
min_closeback = input.float(50.0,"Min close-back of pierce (%)",10, 100, group=gS)
fvg_min_pips  = input.float(3.0,"Min FVG size (pips)",          0, 20, step=0.5, group=gS)

gB = "2. HTF Bias (D1+H4+H1)"
use_bias         = input.bool(true, "Use HTF bias filter", group=gB)
d1_ema_len       = input.int(50,    "D1 EMA length",     group=gB)
h4_ema_fast      = input.int(50,    "H4 EMA fast",       group=gB)
h4_ema_slow      = input.int(200,   "H4 EMA slow",       group=gB)
h1_ema_len       = input.int(50,    "H1 EMA length",     group=gB)
bias_strictness  = input.string("2 of 3", "Bias agreement", options=["all 3","2 of 3","any 1"], group=gB)

gA = "3. AMD Session Filter (Kyiv TZ)"
use_amd        = input.bool(true, "Use AMD session filter", group=gA)
allow_asia     = input.bool(false,"Allow Asia entries",     group=gA)
allow_london   = input.bool(true, "Allow London entries",   group=gA)
allow_ny       = input.bool(true, "Allow NY entries",       group=gA)
asia_start_h   = input.int(2,  "Asia start hour (Kyiv)",    0, 23, group=gA)
asia_end_h     = input.int(8,  "Asia end hour (Kyiv)",      0, 23, group=gA)
london_start_h = input.int(9,  "London start hour (Kyiv)",  0, 23, group=gA)
london_end_h   = input.int(12, "London end hour (Kyiv)",    0, 23, group=gA)
ny_start_h     = input.int(14, "NY start hour (Kyiv)",      0, 23, group=gA)
ny_start_m     = input.int(30, "NY start minute (Kyiv)",    0, 59, group=gA)
ny_end_h       = input.int(19, "NY end hour (Kyiv)",        0, 23, group=gA)

gO = "4. Opens anchors"
use_opens_bonus     = input.bool(true, "Score +1 if sweep hits NYM/WO/MO", group=gO)
open_tolerance_pips = input.float(3.0, "Open tolerance (pips)", 0.5, 20, step=0.5, group=gO)

gE = "5. Entry & Risk"
sl_buf_pips  = input.float(3.0,"SL buffer (pips)",          0, 20, step=0.5, group=gE)
min_rr       = input.float(3.0,"Min RR (hard gate)",        1.0, 10.0, step=0.5, group=gE)
rr5_thresh   = input.float(5.0,"RR threshold for +1 score", 1.0, 10.0, step=0.5, group=gE)
limit_window = input.int(20,   "Retest window bars",        5, 60, group=gE)
risk_full_pct= input.float(1.0,"Full risk % (score ≥5)",    0.1, 3.0, step=0.1, group=gE)
risk_half_pct= input.float(0.5,"Half risk % (score 3-4)",   0.1, 2.0, step=0.1, group=gE)

gSc = "6. 6-Point Scoring (ablation toggles)"
sc_htf_poi      = input.bool(true, "+1 if HTF POI is D1/H4", group=gSc)
sc_open_sweep   = input.bool(true, "+1 if sweep hit key open (NYM/WO/MO)", group=gSc)
sc_smt          = input.bool(true, "+1 if SMT divergence", group=gSc)
sc_trending     = input.bool(true, "+1 if trending profile", group=gSc)
sc_rr5          = input.bool(true, "+1 if RR ≥ 1:5", group=gSc)
sc_same_session = input.bool(true, "+1 if sweep+BOS same session", group=gSc)
score_full_min  = input.int(5, "Score threshold for FULL size", 1, 6, group=gSc)
score_half_min  = input.int(3, "Score threshold for HALF size", 1, 6, group=gSc)

gSmt = "7. SMT Divergence"
use_smt           = input.bool(true, "Use SMT filter", group=gSmt)
smt_corr_symbol   = input.symbol("OANDA:GBPUSD", "SMT correlated symbol", group=gSmt)
smt_corr_lookback = input.int(20, "SMT correlation lookback", 5, 100, group=gSmt)
smt_corr_threshold= input.float(0.5, "SMT corr threshold (below = divergent)", 0.0, 1.0, step=0.05, group=gSmt)

gD = "8. Daily Profile (trending/consolidation)"
use_profile          = input.bool(true, "Use daily profile filter", group=gD)
atr_d_len            = input.int(20, "ATR length (D1)", 5, 100, group=gD)
consol_skip_ratio    = input.float(0.5, "Consolidation skip (range < ratio × ATR)", 0.1, 1.5, step=0.05, group=gD)
trending_bonus_ratio = input.float(0.8, "Trending bonus (range ≥ ratio × ATR)",    0.3, 2.0, step=0.05, group=gD)

gX = "9. Exit Management"
max_hold_bars = input.int(16,  "Max hold bars (15m=4h)", 1, 100, group=gX)
eod_hour_utc  = input.int(19,  "EOD close UTC hour", 0, 23, group=gX)
force_sess_close = input.bool(true, "Force close at session boundary", group=gX)

gG = "10. Risk Guards"
use_daily_stop     = input.bool(true, "Use daily stop", group=gG)
daily_stop_pct     = input.float(-3.0, "Daily stop % (of start-of-day equity)", -10.0, -0.5, step=0.5, group=gG)
use_tilt           = input.bool(true, "Use tilt pause", group=gG)
tilt_consec_loss   = input.int(2, "Consec losses → pause", 1, 5, group=gG)
tilt_pause_bars    = input.int(4, "Tilt pause (15m bars, 4=1h)", 1, 50, group=gG)

gN = "11. News (manual)"
news_block_active = input.bool(false, "News block active (manual toggle)", group=gN)

// ═══════════════════════════════════════════════════════════════
// ░ BLOCK 3 — HTF BIAS (D1 + H4 + H1)
// ═══════════════════════════════════════════════════════════════
d1_ema    = request.security(syminfo.tickerid, "D",  ta.ema(close, d1_ema_len), lookahead=barmerge.lookahead_off)
d1_ema_up = request.security(syminfo.tickerid, "D",  ta.ema(close, d1_ema_len) > ta.ema(close, d1_ema_len)[1], lookahead=barmerge.lookahead_off)

[h4_fast, h4_slow] = request.security(syminfo.tickerid, "240",
     [ta.ema(close, h4_ema_fast), ta.ema(close, h4_ema_slow)],
     lookahead=barmerge.lookahead_off)

[h1_ema_v, h1_close] = request.security(syminfo.tickerid, "60",
     [ta.ema(close, h1_ema_len), close],
     lookahead=barmerge.lookahead_off)

// Per-TF long/short votes
d1_long  = d1_ema_up and close > d1_ema
d1_short = not d1_ema_up and close < d1_ema
h4_long  = h4_fast > h4_slow
h4_short = h4_fast < h4_slow
h1_long  = h1_close > h1_ema_v
h1_short = h1_close < h1_ema_v

long_votes  = (d1_long ? 1 : 0) + (h4_long ? 1 : 0) + (h1_long ? 1 : 0)
short_votes = (d1_short ? 1 : 0) + (h4_short ? 1 : 0) + (h1_short ? 1 : 0)

min_votes = bias_strictness == "all 3" ? 3 : bias_strictness == "2 of 3" ? 2 : 1
bias_long_raw  = long_votes  >= min_votes
bias_short_raw = short_votes >= min_votes
bias_long  = not use_bias or bias_long_raw
bias_short = not use_bias or bias_short_raw

// HTF POI proxy (D1/H4 bias strong = "+1 score если score_htf_poi")
htf_poi_d1h4 = (d1_long and h4_long) or (d1_short and h4_short)

// ═══════════════════════════════════════════════════════════════
// ░ BLOCK 4 — OPENS & LIQUIDITY MAP
// ═══════════════════════════════════════════════════════════════
d_open = request.security(syminfo.tickerid, "D", open,  lookahead=barmerge.lookahead_off)
w_open = request.security(syminfo.tickerid, "W", open,  lookahead=barmerge.lookahead_off)
m_open = request.security(syminfo.tickerid, "M", open,  lookahead=barmerge.lookahead_off)
pdh    = request.security(syminfo.tickerid, "D", high[1], lookahead=barmerge.lookahead_off)
pdl    = request.security(syminfo.tickerid, "D", low[1],  lookahead=barmerge.lookahead_off)
pwh    = request.security(syminfo.tickerid, "W", high[1], lookahead=barmerge.lookahead_off)
pwl    = request.security(syminfo.tickerid, "W", low[1],  lookahead=barmerge.lookahead_off)

// NYM proxy = daily session open o UTC 05:00 (NY midnight ≈ Kyiv 07:00)
var float nym = na
is_nym_bar = hour(time, "UTC") == 5 and minute(time, "UTC") == 0
if is_nym_bar
    nym := open

// Check чи sweep зачепив "ключовий open" (NYM / WO / MO)
tol = open_tolerance_pips * pip
touches_key_open(level) =>
    not na(nym) and math.abs(level - nym) <= tol
     or (not na(w_open) and math.abs(level - w_open) <= tol)
     or (not na(m_open) and math.abs(level - m_open) <= tol)
     or (not na(d_open) and math.abs(level - d_open) <= tol)
     or (not na(pdh)    and math.abs(level - pdh)    <= tol)
     or (not na(pdl)    and math.abs(level - pdl)    <= tol)
     or (not na(pwh)    and math.abs(level - pwh)    <= tol)
     or (not na(pwl)    and math.abs(level - pwl)    <= tol)

// ═══════════════════════════════════════════════════════════════
// ░ BLOCK 5 — DAILY PROFILE (trending vs consolidation)
// ═══════════════════════════════════════════════════════════════
atr_d    = request.security(syminfo.tickerid, "D", ta.atr(atr_d_len), lookahead=barmerge.lookahead_off)
dhigh    = request.security(syminfo.tickerid, "D", high, lookahead=barmerge.lookahead_off)
dlow     = request.security(syminfo.tickerid, "D", low,  lookahead=barmerge.lookahead_off)
day_rng  = not na(dhigh) and not na(dlow) ? dhigh - dlow : na
rng_ratio = not na(day_rng) and not na(atr_d) and atr_d > 0 ? day_rng / atr_d : na

is_consol_day   = use_profile and not na(rng_ratio) and rng_ratio < consol_skip_ratio
is_trending_day = not na(rng_ratio) and rng_ratio >= trending_bonus_ratio

// ═══════════════════════════════════════════════════════════════
// ░ BLOCK 6 — AMD SESSION STATE + session_id
// ═══════════════════════════════════════════════════════════════
min_kyiv = f_bar_time_minutes_kyiv()
in_asia   = min_kyiv >= asia_start_h   * 60 and min_kyiv < asia_end_h   * 60
in_london = min_kyiv >= london_start_h * 60 and min_kyiv < london_end_h * 60
in_ny     = min_kyiv >= ny_start_h     * 60 + ny_start_m and min_kyiv < ny_end_h * 60

session_allowed =
     (in_asia   and allow_asia)
     or (in_london and allow_london)
     or (in_ny     and allow_ny)
in_session = not use_amd or session_allowed

// session_id: монотонний лічильник — incremented on session start transitions
var int  session_id         = 0
var bool in_any_session_prev = false
in_any_session = in_asia or in_london or in_ny
if in_any_session and not in_any_session_prev
    session_id += 1
in_any_session_prev := in_any_session

// ═══════════════════════════════════════════════════════════════
// ░ BLOCK 7 — PIVOTS
// ═══════════════════════════════════════════════════════════════
buf         = sl_buf_pips * pip
min_pierce_px = min_pierce * pip

ph = ta.pivothigh(high, piv_len, piv_len)
pl = ta.pivotlow(low,   piv_len, piv_len)

var float last_ph    = na
var float last_pl    = na
var int   last_ph_bi = na
var int   last_pl_bi = na

if not na(ph)
    last_ph    := ph
    last_ph_bi := bar_index - piv_len
if not na(pl)
    last_pl    := pl
    last_pl_bi := bar_index - piv_len

// ═══════════════════════════════════════════════════════════════
// ░ BLOCK 8 — SWEEP DETECTION (+ session_id at sweep)
// ═══════════════════════════════════════════════════════════════
var bool  bull_sw       = false
var float bull_sw_low   = na
var int   bull_sw_bi    = na
var int   bull_sw_sid   = na
var bool  bull_sw_keyopen = false

var bool  bear_sw       = false
var float bear_sw_high  = na
var int   bear_sw_bi    = na
var int   bear_sw_sid   = na
var bool  bear_sw_keyopen = false

// Bullish sweep: low pierces last_pl, closes back above
if not na(last_pl) and (bar_index - last_pl_bi) <= sweep_lb
    pierce = last_pl - low
    closeback_dist = close - low
    if low < last_pl and close > last_pl and pierce >= min_pierce_px and closeback_dist >= pierce * (min_closeback / 100)
        bull_sw         := true
        bull_sw_low     := low
        bull_sw_bi      := bar_index
        bull_sw_sid     := session_id
        bull_sw_keyopen := touches_key_open(last_pl)

// Bearish sweep: high pierces last_ph, closes back below
if not na(last_ph) and (bar_index - last_ph_bi) <= sweep_lb
    pierce = high - last_ph
    closeback_dist = high - close
    if high > last_ph and close < last_ph and pierce >= min_pierce_px and closeback_dist >= pierce * (min_closeback / 100)
        bear_sw         := true
        bear_sw_high    := high
        bear_sw_bi      := bar_index
        bear_sw_sid     := session_id
        bear_sw_keyopen := touches_key_open(last_ph)

// ═══════════════════════════════════════════════════════════════
// ░ BLOCK 9 — LTF BOS (after sweep)
// ═══════════════════════════════════════════════════════════════
var bool  bull_bos       = false
var float bull_bos_lvl   = na
var int   bull_bos_bi    = na
var int   bull_bos_sid   = na

var bool  bear_bos       = false
var float bear_bos_lvl   = na
var int   bear_bos_bi    = na
var int   bear_bos_sid   = na

if bull_sw and not na(last_ph) and close > last_ph
    bull_bos     := true
    bull_bos_lvl := last_ph
    bull_bos_bi  := bar_index
    bull_bos_sid := session_id
    bull_sw      := false

if bear_sw and not na(last_pl) and close < last_pl
    bear_bos     := true
    bear_bos_lvl := last_pl
    bear_bos_bi  := bar_index
    bear_bos_sid := session_id
    bear_sw      := false

if bull_bos and (bar_index - bull_bos_bi) > limit_window
    bull_bos := false
if bear_bos and (bar_index - bear_bos_bi) > limit_window
    bear_bos := false

// ═══════════════════════════════════════════════════════════════
// ░ BLOCK 10 — LTF ENTRY MODEL (FVG proximal check)
// ═══════════════════════════════════════════════════════════════
// 3-bar bullish FVG: high[2] < low[0] → gap.  Proximal = high[2], distal = low[0]
// 3-bar bearish FVG: low[2]  > high[0]
bull_fvg_prox = high[2] < low[0] and (low[0] - high[2]) >= fvg_min_pips * pip ? high[2] : na
bear_fvg_prox = low[2]  > high[0] and (low[2] - high[0]) >= fvg_min_pips * pip ? low[2]  : na

// ═══════════════════════════════════════════════════════════════
// ░ BLOCK 11 — SMT DIVERGENCE (proxy)
// ═══════════════════════════════════════════════════════════════
smt_close = request.security(smt_corr_symbol, timeframe.period, close, lookahead=barmerge.lookahead_off)
smt_corr  = not na(smt_close) ? ta.correlation(close, smt_close, smt_corr_lookback) : na
smt_divergent = use_smt and not na(smt_corr) and smt_corr < smt_corr_threshold

// ═══════════════════════════════════════════════════════════════
// ░ BLOCK 12 — 5-STEP FUNNEL BOOLEANS
// ═══════════════════════════════════════════════════════════════
// For LONG side we evaluate on the bar where bull_bos was just set
long_step1 = bias_long
long_step2 = htf_poi_d1h4                 // proxy: HTF POI means D1+H4 agreeing
long_step3 = bull_bos and bull_sw_bi != na    // sweep already happened
long_step4 = bull_bos                     // BOS up confirmed
long_step5 = not na(bull_fvg_prox) or true   // FVG optional — fallback to BOS level entry
long_all_5 = long_step1 and long_step2 and long_step3 and long_step4 and long_step5

short_step1 = bias_short
short_step2 = htf_poi_d1h4
short_step3 = bear_bos and bear_sw_bi != na
short_step4 = bear_bos
short_step5 = not na(bear_fvg_prox) or true
short_all_5 = short_step1 and short_step2 and short_step3 and short_step4 and short_step5

// ═══════════════════════════════════════════════════════════════
// ░ BLOCK 13 — 6-POINT SCORING
// ═══════════════════════════════════════════════════════════════
calc_score_long() =>
    s = 0
    s += (sc_htf_poi      and htf_poi_d1h4)                      ? 1 : 0
    s += (sc_open_sweep   and bull_sw_keyopen)                   ? 1 : 0
    s += (sc_smt          and smt_divergent)                     ? 1 : 0
    s += (sc_trending     and is_trending_day)                   ? 1 : 0
    // RR ≥ 1:5 check: evaluate with provisional risk/TP (computed later); approximated here via pivot distance
    rr_possible = not na(last_ph) and not na(bull_sw_low) ? (last_ph - bull_bos_lvl) / math.max(bull_bos_lvl - bull_sw_low, syminfo.mintick) : 0.0
    s += (sc_rr5          and rr_possible >= rr5_thresh)         ? 1 : 0
    s += (sc_same_session and bull_sw_sid == bull_bos_sid)       ? 1 : 0
    s

calc_score_short() =>
    s = 0
    s += (sc_htf_poi      and htf_poi_d1h4)                      ? 1 : 0
    s += (sc_open_sweep   and bear_sw_keyopen)                   ? 1 : 0
    s += (sc_smt          and smt_divergent)                     ? 1 : 0
    s += (sc_trending     and is_trending_day)                   ? 1 : 0
    rr_possible = not na(last_pl) and not na(bear_sw_high) ? (bear_bos_lvl - last_pl) / math.max(bear_sw_high - bear_bos_lvl, syminfo.mintick) : 0.0
    s += (sc_rr5          and rr_possible >= rr5_thresh)         ? 1 : 0
    s += (sc_same_session and bear_sw_sid == bear_bos_sid)       ? 1 : 0
    s

score_long_val  = calc_score_long()
score_short_val = calc_score_short()

size_mult_long  = score_long_val  >= score_full_min ? 1.0 : score_long_val  >= score_half_min ? 0.5 : 0.0
size_mult_short = score_short_val >= score_full_min ? 1.0 : score_short_val >= score_half_min ? 0.5 : 0.0

// ═══════════════════════════════════════════════════════════════
// ░ BLOCK 14 — RISK GUARDS (daily stop + tilt pause)
// ═══════════════════════════════════════════════════════════════
// Start-of-day equity tracking
var float sod_equity = na
is_new_day = ta.change(dayofweek(time, "UTC")) != 0
if is_new_day or na(sod_equity)
    sod_equity := strategy.equity
day_pnl_pct = sod_equity > 0 ? (strategy.equity - sod_equity) / sod_equity * 100 : 0.0
daily_stop_hit = use_daily_stop and day_pnl_pct <= daily_stop_pct

// Consec loss tracker
var int  consec_losses  = 0
var int  tilt_until_bi  = 0
if strategy.closedtrades > strategy.closedtrades[1]
    last_pnl = strategy.closedtrades.profit(strategy.closedtrades - 1)
    if last_pnl < 0
        consec_losses += 1
        if use_tilt and consec_losses >= tilt_consec_loss
            tilt_until_bi := bar_index + tilt_pause_bars
            consec_losses := 0
    else
        consec_losses := 0
tilt_active = bar_index < tilt_until_bi

// ═══════════════════════════════════════════════════════════════
// ░ BLOCK 15 — HARD GATES & ENTRY
// ═══════════════════════════════════════════════════════════════
base_gates_ok =
     in_session
     and not is_consol_day
     and not news_block_active
     and not daily_stop_hit
     and not tilt_active
     and strategy.position_size == 0

// LONG entry
if base_gates_ok and long_all_5 and size_mult_long > 0.0
    entry_px = not na(bull_fvg_prox) ? bull_fvg_prox : bull_bos_lvl
    sl_px    = bull_sw_low - buf
    risk     = entry_px - sl_px
    if risk > 0
        target_liq = na(last_ph) ? entry_px + risk * min_rr : math.max(last_ph, entry_px + risk * min_rr)
        rr_to_liq  = (target_liq - entry_px) / risk
        if rr_to_liq >= min_rr
            risk_pct = risk_full_pct * size_mult_long
            risk_money = strategy.equity * (risk_pct / 100.0)
            qty = risk_money / risk
            strategy.entry("L", strategy.long, qty=qty, limit=entry_px)
            strategy.exit("XL", "L", stop=sl_px, limit=target_liq)
            bull_bos := false

// SHORT entry
if base_gates_ok and short_all_5 and size_mult_short > 0.0
    entry_px = not na(bear_fvg_prox) ? bear_fvg_prox : bear_bos_lvl
    sl_px    = bear_sw_high + buf
    risk     = sl_px - entry_px
    if risk > 0
        target_liq = na(last_pl) ? entry_px - risk * min_rr : math.min(last_pl, entry_px - risk * min_rr)
        rr_to_liq  = (entry_px - target_liq) / risk
        if rr_to_liq >= min_rr
            risk_pct = risk_full_pct * size_mult_short
            risk_money = strategy.equity * (risk_pct / 100.0)
            qty = risk_money / risk
            strategy.entry("S", strategy.short, qty=qty, limit=entry_px)
            strategy.exit("XS", "S", stop=sl_px, limit=target_liq)
            bear_bos := false

// ═══════════════════════════════════════════════════════════════
// ░ BLOCK 16 — EXIT MANAGEMENT (EOD + session boundary + time stop)
// ═══════════════════════════════════════════════════════════════
bars_in_pos = strategy.position_size != 0 ? bar_index - strategy.opentrades.entry_bar_index(0) : 0
time_stop_hit = strategy.position_size != 0 and bars_in_pos >= max_hold_bars
eod_hit = strategy.position_size != 0 and hour(time, "UTC") >= eod_hour_utc
sess_boundary_close = force_sess_close and strategy.position_size != 0 and in_any_session_prev and not in_any_session

if time_stop_hit
    strategy.close_all(comment="time-stop")
if eod_hit
    strategy.close_all(comment="EOD")
if sess_boundary_close
    strategy.close_all(comment="session-end")

if hour(time, "UTC") >= eod_hour_utc
    strategy.cancel_all()

// ═══════════════════════════════════════════════════════════════
// ░ BLOCK 17 — VISUALS
// ═══════════════════════════════════════════════════════════════
plotshape(bull_bos and not bull_bos[1], "BOS↑", shape.triangleup,   location.belowbar, color.lime, size=size.tiny)
plotshape(bear_bos and not bear_bos[1], "BOS↓", shape.triangledown, location.abovebar, color.red,  size=size.tiny)
plotshape(bull_sw  and not bull_sw[1],  "SW↑",  shape.circle,       location.belowbar, color.new(color.lime, 50), size=size.tiny)
plotshape(bear_sw  and not bear_sw[1],  "SW↓",  shape.circle,       location.abovebar, color.new(color.red,  50), size=size.tiny)

bgcolor(in_asia   ? color.new(color.blue,   92) : na, title="Asia")
bgcolor(in_london ? color.new(color.orange, 92) : na, title="London")
bgcolor(in_ny     ? color.new(color.green,  92) : na, title="NY")

plot(use_bias ? h1_ema_v : na, "H1 EMA", color.orange, 1)

// Score label on fresh BOS
if bull_bos and not bull_bos[1]
    label.new(bar_index, low, "L score=" + str.tostring(score_long_val),
         style=label.style_label_up, color=color.new(color.lime, 60), textcolor=color.black, size=size.tiny)
if bear_bos and not bear_bos[1]
    label.new(bar_index, high, "S score=" + str.tostring(score_short_val),
         style=label.style_label_down, color=color.new(color.red, 60), textcolor=color.white, size=size.tiny)
```

## ⚠️ Pine limitations vs manual NS

Pine **НЕ моделює:**
- **OB validity grading** — algorithmic proxy (pivot + imbalance) замість discretionary OB selection
- **FVG type** (IOFED vs 50% vs full fill) — всі FVG однакові
- **Discount/premium (OTE zone)** — не враховується HTF range context
- **Kyiv DST shoulder weeks** — `"Europe/Kiev"` tz string обробляє DST автоматично, але sweep/BOS detection може зміщуватись ±1h у transition тижні
- **News calendar** — тільки ручний `news_block_active` toggle
- **Correlation cap (EUR+GBP no double long)** — N/A на single-pair backtest; тест cross-pair робиться окремими run
- **Psychology / emotional tilt** — тільки mechanical consec-loss counter
- **Expert sweep quality judgment** — тільки pierce+closeback thresholds
- **SMT divergence** — проксі через `ta.correlation < threshold`; реальний SMT — non-confirmation swing extremes, що не піддається простій Pine-формулі

**Висновок:** Pine = **lower bound edge estimate**. Якщо Pine даєш позитивний результат → manual NS має дати ще кращий. Якщо Pine у збитку → discretionary частина може все ж мати edge (урок з ТС-2/ТС-3), але слабкий сигнал.
