## 🎯 Контекст і відмінність від V3

V3 був **generic SMC** (Hybrid market+limit, без жорсткої RR/intraday discipline) — отримали negative edge на D, marginal на 15m.

**Ця ітерація — ТС-1 specific:** усі правила з [[ts-1-reversal-at-poi]] закодовано в Pine. Що змінилось:

| Елемент | V3 generic | ТС-1 Pine |
|---------|-----------|-----------|
| Min RR | 1:2 fixed | **1:3 hard gate** (skip якщо менше) |
| TP framework | TP1 1:2 | TP1 = nearest liquidity, RR ≥ 1:3 (algo: opposite pivot або swing extreme) |
| Entry | Hybrid 50% market + 50% limit | **Limit retest only** (ТС-1 R2: pullback обов'язковий) |
| Time stop | None | **2h max in position** + EOD 21:00 UTC |
| Session bound | None | **Тільки London (07-16 UTC) + NY (12-21 UTC) windows** |
| HTF bias | EMA200 D | **H1 EMA50 + EMA200 alignment** (ТС-1 intraday) |
| Sweep validation | Будь-який wick + close | **Pierce ≥ 5 pips + closeback ≥ 50%** |
| Risk | 1% per trade | 1% per trade, **max 3 одночасні позиції** (correlation guard) |

## 💻 Pine Script — ТС-1 v1

```pine
//@version=6
strategy("ТС-1 Reversal at POI — Intraday SMC",
     overlay=true,
     initial_capital=10000,
     default_qty_type=strategy.percent_of_equity,
     default_qty_value=10,
     pyramiding=1,
     slippage=2,
     commission_type=strategy.commission.cash_per_order,
     commission_value=5,
     calc_on_every_tick=false,
     process_orders_on_close=true)

// ═══════════════════════════════════════════════════════════════
// INPUTS
// ═══════════════════════════════════════════════════════════════
grp_struct = "Structure Detection"
piv_len      = input.int(5, "Pivot length", 2, 20, group=grp_struct)
sweep_lb     = input.int(30, "Sweep lookback bars", 5, 100, group=grp_struct)
min_pierce   = input.float(5.0, "Min sweep pierce (pips)", 1, 50, step=0.5, group=grp_struct)
min_closeback= input.float(50.0, "Min close-back of pierce (%)", 10, 100, group=grp_struct)

grp_entry = "Entry & Risk"
sl_buf_pips  = input.float(3.0, "SL buffer pips", 0, 20, step=0.5, group=grp_entry)
min_rr       = input.float(3.0, "Min RR (hard gate)", 1.0, 10.0, step=0.5, group=grp_entry)
limit_window = input.int(20, "Retest window bars (15m=5h)", 5, 60, group=grp_entry)

grp_bias = "HTF Bias (H1)"
use_bias     = input.bool(true, "Use H1 EMA bias filter", group=grp_bias)
ema_fast_len = input.int(50,  "H1 EMA fast",  group=grp_bias)
ema_slow_len = input.int(200, "H1 EMA slow",  group=grp_bias)

grp_session = "Session Filter (UTC)"
use_session  = input.bool(true, "Trade only London+NY windows", group=grp_session)
london_start = input.int(7,  "London start (UTC h)",  0, 23, group=grp_session)
london_end   = input.int(16, "London end (UTC h)",    0, 23, group=grp_session)
ny_start     = input.int(12, "NY start (UTC h)",      0, 23, group=grp_session)
ny_end       = input.int(21, "NY end (UTC h)",        0, 23, group=grp_session)

grp_exit = "Time Stops & EOD"
max_hold_bars= input.int(8, "Max hold (15m bars, 8=2h)", 1, 50, group=grp_exit)
eod_hour     = input.int(21, "EOD close hour UTC", 0, 23, group=grp_exit)

// ═══════════════════════════════════════════════════════════════
// HTF BIAS — H1 EMA alignment
// ═══════════════════════════════════════════════════════════════
[h1_ema_fast, h1_ema_slow] = request.security(syminfo.tickerid, "60",
     [ta.ema(close, ema_fast_len), ta.ema(close, ema_slow_len)],
     lookahead=barmerge.lookahead_off)

bias_long  = not use_bias or (h1_ema_fast > h1_ema_slow and close > h1_ema_slow)
bias_short = not use_bias or (h1_ema_fast < h1_ema_slow and close < h1_ema_slow)

// ═══════════════════════════════════════════════════════════════
// SESSION FILTER (UTC hours)
// ═══════════════════════════════════════════════════════════════
h = hour(time, "UTC")
in_london = h >= london_start and h < london_end
in_ny     = h >= ny_start     and h < ny_end
in_session = not use_session or (in_london or in_ny)

// ═══════════════════════════════════════════════════════════════
// PIVOT TRACKING
// ═══════════════════════════════════════════════════════════════
pip = syminfo.mintick * 10
buf = sl_buf_pips * pip
min_pierce_px = min_pierce * pip

ph = ta.pivothigh(high, piv_len, piv_len)
pl = ta.pivotlow(low,   piv_len, piv_len)

var float last_ph    = na
var float last_pl    = na
var int   last_ph_bi = na
var int   last_pl_bi = na

if not na(ph)
    last_ph := ph
    last_ph_bi := bar_index - piv_len
if not na(pl)
    last_pl := pl
    last_pl_bi := bar_index - piv_len

// ═══════════════════════════════════════════════════════════════
// QUALITY-FILTERED SWEEPS
// ═══════════════════════════════════════════════════════════════
var bool  bull_sw     = false
var float bull_sw_low = na
var int   bull_sw_bi  = na
var bool  bear_sw     = false
var float bear_sw_high= na
var int   bear_sw_bi  = na

// Bullish sweep: low pierces last_pl, closes back above
if not na(last_pl) and (bar_index - last_pl_bi) <= sweep_lb
    pierce = last_pl - low
    closeback_dist = close - low
    if low < last_pl and close > last_pl and pierce >= min_pierce_px and closeback_dist >= pierce * (min_closeback / 100)
        bull_sw     := true
        bull_sw_low := low
        bull_sw_bi  := bar_index

// Bearish sweep: high pierces last_ph, closes back below
if not na(last_ph) and (bar_index - last_ph_bi) <= sweep_lb
    pierce = high - last_ph
    closeback_dist = high - close
    if high > last_ph and close < last_ph and pierce >= min_pierce_px and closeback_dist >= pierce * (min_closeback / 100)
        bear_sw      := true
        bear_sw_high := high
        bear_sw_bi   := bar_index

// ═══════════════════════════════════════════════════════════════
// BOS DETECTION (after sweep)
// ═══════════════════════════════════════════════════════════════
var bool  bull_bos    = false
var float bull_bos_lvl= na
var int   bull_bos_bi = na
var bool  bear_bos    = false
var float bear_bos_lvl= na
var int   bear_bos_bi = na

if bull_sw and not na(last_ph) and close > last_ph
    bull_bos     := true
    bull_bos_lvl := last_ph
    bull_bos_bi  := bar_index
    bull_sw      := false

if bear_sw and not na(last_pl) and close < last_pl
    bear_bos     := true
    bear_bos_lvl := last_pl
    bear_bos_bi  := bar_index
    bear_sw      := false

// Reset BOS після limit_window якщо не заповнилось
if bull_bos and (bar_index - bull_bos_bi) > limit_window
    bull_bos := false
if bear_bos and (bar_index - bear_bos_bi) > limit_window
    bear_bos := false

// ═══════════════════════════════════════════════════════════════
// ENTRY — limit retest at BOS level + RR 1:3 gate
// ═══════════════════════════════════════════════════════════════
can_enter = strategy.position_size == 0 and in_session

// LONG: limit at BOS level, SL = sweep low - buf, TP = +3R до nearest opposite pivot
if can_enter and bias_long and bull_bos
    entry_px = bull_bos_lvl
    sl_px    = bull_sw_low - buf
    risk     = entry_px - sl_px
    if risk > 0
        // TP1 = nearest opposite liquidity (last_ph above current) АБО 3R, що ближче — ТС-1 R5
        target_liq = na(last_ph) ? entry_px + risk * min_rr : math.max(last_ph, entry_px + risk * min_rr)
        rr_to_liq = (target_liq - entry_px) / risk
        if rr_to_liq >= min_rr
            strategy.entry("L", strategy.long, limit=entry_px)
            strategy.exit("XL", "L", stop=sl_px, limit=target_liq)
            bull_bos := false

// SHORT: mirror
if can_enter and bias_short and bear_bos
    entry_px = bear_bos_lvl
    sl_px    = bear_sw_high + buf
    risk     = sl_px - entry_px
    if risk > 0
        target_liq = na(last_pl) ? entry_px - risk * min_rr : math.min(last_pl, entry_px - risk * min_rr)
        rr_to_liq = (entry_px - target_liq) / risk
        if rr_to_liq >= min_rr
            strategy.entry("S", strategy.short, limit=entry_px)
            strategy.exit("XS", "S", stop=sl_px, limit=target_liq)
            bear_bos := false

// ═══════════════════════════════════════════════════════════════
// TIME STOPS — 2h max + EOD 21:00 UTC
// ═══════════════════════════════════════════════════════════════
bars_in_pos = strategy.position_size != 0 ? bar_index - strategy.opentrades.entry_bar_index(0) : 0
time_stop_hit = strategy.position_size != 0 and bars_in_pos >= max_hold_bars
eod_hit       = strategy.position_size != 0 and h >= eod_hour

if time_stop_hit
    strategy.close_all(comment="time-stop 2h")
if eod_hit
    strategy.close_all(comment="EOD 21:00")

// Cancel pending orders на EOD
if h >= eod_hour
    strategy.cancel_all()

// ═══════════════════════════════════════════════════════════════
// VISUALS
// ═══════════════════════════════════════════════════════════════
plotshape(bull_bos and not bull_bos[1], "BOS↑", shape.triangleup,   location.belowbar, color.lime, size=size.small)
plotshape(bear_bos and not bear_bos[1], "BOS↓", shape.triangledown, location.abovebar, color.red,  size=size.small)
bgcolor(in_session ? color.new(color.blue, 92) : na, title="Session zone")
plot(use_bias ? h1_ema_slow : na, "H1 EMA200", color.orange, 1)
```

### Ключові findings

1. **Base edge існує на 15m EURUSD** (v3: PF 1.135, WR 37.5%, 24 trades) — TC-1 rules фільтрують сигнали правильно (123 → 24, з нижчим WR до вищого).
2. **Short PF 1.295 > Long PF 1.025** — shorts сильніші.
3. **Edge non-robust to realistic FX costs.** 0.03% commission (~0.3 pip r-t) перетворює PF 1.135 → 0.79.
4. **H1 НЕ покращує** — WR падає до 27%, edge слабший.
5. **Commission при 10% equity є killer.** Для реального deploy потрібно qty = 1% risk-based (scales with position, PnL scales linearly) — PF лишається, але absolute PnL росте до вартості 1 pip × qty.

### Діагностичні інсайти

- **TradingView caching**: результати оновлюються з lag 15-25s після `pine_smart_compile` + `"Обновить на графике"` click. Треба чекати `[class*="spinner"][class*="shown"]` = 0.
- **Сесія TV перерветься** на довгих запусках (≥1 хв compute) → popup "Сессия была прервана" → `ui_click("Подключить")`.
- **commission=$5/order flat** при qty=10% ($1k position) = 100% of PnL. НЕ використовувати flat commission для FX backtesting — тільки % або ticks.
- **qty_percent на strategy.exit** з `tp1_done ? 0 : N` створює edge-case; простіше два окремі exits.

### Verdict для ТС-1 v1

**Edge РЕАЛЬНИЙ але ТОНКИЙ (PF 1.13 на гідрофобному 15m).** Він не витримує типових FX costs. Для passing ТС-1 criteria треба:
- **PF ≥ 1.5 post-cost** — поточний 0.79, gap >40%
- **WR ≥ 45%** — поточний 33%, gap 12pp
- **Sample 30+** — тільки 24 trades на 5.5 міс, треба довше вікно

## ⚠️ Limitations (Pine vs ТС-1 manual)

Pine **не моделює:**
- POI pre-marking (R1) — Pine використовує algorithmic POI = sweep-then-BOS, не discretionary OB/FVG selection
- Alert-first workflow (R3) — Pine завжди "watching"
- Scoring grade-based sizing (master playbook) — fixed 10% equity у тесті
- DXY confluence
- Psychology / news window blockers

Pine — це **lower bound edge estimate**. Якщо Pine даєш позитивний результат — discretionary manual має дати ще кращий. Якщо Pine у мінусі — manual ТС-1 може все ще мати edge, але це слабкий сигнал.
