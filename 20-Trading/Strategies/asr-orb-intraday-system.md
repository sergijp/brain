## Стратегія №1 — Asia Sweep & Reclaim (ядро)

**Edge:** азійський діапазон = скупчення стопів; London open робить помилковий пробій (Judas swing), знімає ліквідність і повертається. Торгуємо провалений пробій (sweep-and-fail).

### Псевдокод

```
ПАРАМЕТРИ:
  ATR_D = ATR(14) Daily (вчорашнє закриття), ATR_SMA = SMA(ATR_D, 50)
  TF = M15, TZ = Europe/Kiev (явно у кожному time())

ASIA RANGE:
  EURUSD/GBPUSD: бари [03:00, 09:00) Київ
  GER40:         бари [22:00, 09:00) Київ (той самий CFD-символ, 24h-фід, нуль request.security)
  AH = max(high), AL = min(low); RW = AH − AL

ДЕННІ ФІЛЬТРИ (один раз; FX о 09:00, GER40 о 10:05):
  F1: 0.20×ATR_D ≤ RW ≤ 0.60×ATR_D
  F2: ATR_D ≥ 0.60×ATR_SMA
  F3: новинна політика PASS (див. розділ Новини)
  F4: вимикач трендової серії OFF (2 поспіль дні |C−O| > 0.7×ATR_D однакового знаку → №1 off наступного дня)
  F5: allowed_dir: вчора |C−O| > 0.7×ATR_D → тільки напрямок sign(C−O) вчора; інакше both
  будь-який FAIL → №1 неактивна сьогодні на інструменті

ВІКНО SWEEP: FX [09:00, 13:00); GER40 [10:05, 13:00) (жодних входів до 10:05 — pre-Xetra спред)
P2 (патч Kassandra): якщо о 10:05 ціна GER40 ВЖЕ за Asia-екстремумом (пробій стався у сліпій зоні
09:00–10:05 або через геп) → сетап на цю сторону SKIP на день. Бар 0 завжди всередині вікна.

BUFFER = max(0.05×ATR_D, floor); floor = 3 pips FX, 12 pt GER40

SWEEP (LONG; SHORT дзеркально):
  бар i: low(i) < AL − BUFFER → sweep_bar = i; sweep_low оновлюється, поки ціна нижче AL
RECLAIM:
  бар j ∈ [i, i+4] (бар пробою = бар 0): close(j) > AL → reclaim_bar = j; інакше сетап скасовано
ВХІД: buy-stop @ high(reclaim_bar) + 1 тік, валідність 2 бари M15
SL:  sweep_low − BUFFER
TP1: AH (закрити 50%, решта SL→BE)
TP2: варіант A = AH + 0.5×RW; варіант B = AH + 1.0×RW — ОБИДВА у бектест, рішення за фактичним p_full (P4)
PRE-TRADE GATE: RR(entry→TP2 vs entry→SL) ≥ 1.8, інакше skip
ЛІМІТИ: 1 вхід/напрямок/день; max 2 сетапи/день/інструмент (re-entry/повторне озброєння РАХУЄТЬСЯ у ліміт);
        2 стопи поспіль на інструменті → off до завтра
FLAT 17:45 Київ
```

## Стратегія №2 — Xetra ORB-30 (тільки GER40, satellite)

**Edge:** пробій 30-хв opening range Xetra з продовженням (Initial Balance).

```
OR: ORH/ORL барів [10:00, 10:30) Київ (Pine session timezone="Europe/Berlin" 0900-0930); ORW = ORH − ORL
ФІЛЬТРИ (о 10:30): F1: 0.10×ATR_D ≤ ORW ≤ 0.30×ATR_D
  F2 gap: |Open(10:00 Київ CFD) − Close(18:30 Київ попер. дня, той самий фід)| ≤ 0.30×ATR_D
  F3 новини PASS; F4 GER40-lock вільний (G4)
ОРДЕРИ (10:30): OCO buy-stop @ ORH+1 тік / sell-stop @ ORL−1 тік; одна сторона виконалась → друга cancel негайно; до 13:00
SL: протилежний край OR
TP1: 1×ORW (50%, SL→BE); TP2: max(2×ORW, 0.5×ATR_D)
PRE-TRADE GATE: RR(TP2) ≥ 1.8 на 10:30, інакше OCO не ставиться
RE-ENTRY: після стоп-ауту одне повторне OCO до 13:00; після 2-го стопу done; cancel за часом ре-вхід не дає
БАР-ГОНКА: бектест bar magnifier M1 / fallback «стоп перший»; бот server-side OCO; signal-mode: двотригерний бар = no-trade
FLAT 17:45
```

- Трейлінг 3-бари — **видалено у v1** (повернеться тільки у боті v2).
- Signal-mode: 2 напівпозиції з одним SL, одна ручна дія (BE після TP1).

## Risk-Governor (глобальний, поверх обох стратегій)

```
G1 HARD-STOP: realized_day_loss + open_risk ≥ 3.0% → cancel all pending, нових входів немає до завтра
G2 CONCURRENT: сумарний open_risk ≤ 2% (max 2 не-BE позиції)
G3 ONE-IDEA USD: EURUSD+GBPUSD = одна ідея. GBPUSD-сетап тільки якщо EURUSD без позиції/pending у тому ж напрямку.
   Одночасний сигнал (±30 хв) → береться EURUSD (вужчий спред), GBPUSD skip.
P1 G3-SEQUENTIAL (патч): стоп на EURUSD у напрямку X сьогодні → GBPUSD напрямок X off до завтра (і дзеркально).
   Одна ідея = одна спроба.
G4 GER40-LOCK: перша з {№1, №2}, що озброїла ордер, лочить GER40 на день
G5 MACRO-CLUSTER: 2 стопи по №1 до 13:00 (сумарно всі інструменти) → №1 off до завтра
G6 PER-INSTRUMENT: 2 стопи поспіль → інструмент off до завтра
G7 FLAT 17:45; п'ятниця після 17:00 — без нових входів
САЙЗИНГ: flat 1.0% = $100 / стоп-дистанція. Без винятків.
```
